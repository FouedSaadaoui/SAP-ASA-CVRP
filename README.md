# SAP-ASA-CVRP

A data-driven adaptive Simulated Annealing framework for the Capacitated Vehicle Routing Problem (CVRP), integrating stratified acceptance probabilities and online Weibull mixture modeling of cost increments.

## Related Work

This repository is associated with the research article:

"Stratified Acceptance-Probability-based Adaptive Simulated Annealing for Efficient Optimization: Applications in Econometrics and Graph Analytics", Journal of the Operational Research Society (JORS).

The code provides implementation and experimental support for the adaptive acceptance mechanisms introduced in the paper.
